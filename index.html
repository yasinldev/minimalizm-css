Introduction

One-to-one for medium-sized projects, extensible, and feature-packed frontend toolkit. Build and customize with Pure CSS, utilize prebuilt grid system and components, and bring projects to life with powerful Bootstrap JS plugins.

Support us   

This is a beta version of Minimalizm CSS. Please report any bugs or issues you find.
